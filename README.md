### PC端练习项目

+ 仿制网易云界面制作的web端音乐播放器
+ 使用HTML5,CSS3相关技术

### 功能列表
+ 播放音乐
+ 切换播放模式
+ 显示（关闭）操作栏
+ 歌词滚动
+ 播放列表
